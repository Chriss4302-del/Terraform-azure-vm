# Terraform Azure VM

This project is about creating a virtual machine in Azure using Terraform.

The goal was to understand how infrastructure can be defined and deployed using code instead of setting everything up manually.

## What I did

- used Terraform to create a resource group
- created a virtual network and subnet
- created a Linux VM
- configured basic network access
- deployed everything using Terraform commands

## Why I made this

After setting up a VM manually in Azure, I wanted to try doing the same thing using Infrastructure as Code.

This made it easier to see how setups can be repeated and managed in a more structured way.

## What I learned

- how Terraform works
- how to define infrastructure in code
- how resources are connected in Azure
- why automation is useful in cloud environments
